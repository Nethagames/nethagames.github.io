Node Software

Node Software est un groupe que j'ai cofondé avec un ami qui comporte aujourd'hui une dizaine de membres. Le groupe participe à des game jams et réalise des jeux vidéo indépendants. Chacun possède son rôle :

  • Game Designer
  • Developer
  • Graphist
  • Sound Designer
  • Writer

Game Jams


Patch notes v1.0

Notre premiere jam, d'une durée de 3 jours publiée sur itch.io par indieformer qui regroupait pas moins de 3500 personnes ! Le jeu s'appelle Bastion, le thème était "bugs are the feature" donc on a décidé de créer un jeu, et plutôt que de corriger les bugs rencontrés, axer le gameplay autour de ceux-ci. Scream Jam 2025

Notre deuxième jam, d'une durée d'une semaine cette fois organisée par SCREAM ZONE avec 1000 participations. Le jeu s'appelle HeartLight, le thème était "Horreur" et le jeu que l'on a fait est un platformer 2d ou le but est de fuir un monstre à travers des galeries, l'ambiance a été la priorité durant ce projet donc on s'est concentré sur les visuels et les sons. Godot (&Beyond) Official Awards: The Indie Spark
